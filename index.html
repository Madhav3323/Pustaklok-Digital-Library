üöÄ Project Overview

Pustaklok is a robust two-part application designed for efficiently managing a media collection (Books, Films, Magazines). It utilizes a modern **Flask API backend** for data handling and a user-friendly **Tkinter GUI frontend** for interaction.

This clean, decoupled architecture ensures a maintainable and scalable system, separating data logic from the user interface for optimal performance and reliability.

‚öôÔ∏è Backend API (Flask)

The core logic is managed by a simple **RESTful API** built with Flask (`app.py`). It ensures transactional integrity for all data operations, including:

  • GET /media: Retrieve all items.
  • POST /media: Create a new item.
  • DELETE /media/<name>: Remove an item.
  • POST /media/<name>/borrow: Update item status.

‚ñ∂Ô∏è How to Run Pustaklok

To get the Pustaklok Digital Library up and running, follow these four simple steps:

1.

Prerequisites

Ensure you have Python 3.x and the project files (`app.py`, `gui.py`, `storage.py`) downloaded.

2.

Install Dependencies

Install the required libraries (Flask and Requests) in your terminal:

pip install flask requests
3.

Start the Backend API (Server)

Run the Flask server, which will handle all data requests:

python app.py

The server will run on http://127.0.0.1:5000.

4.

Launch the Frontend GUI

Open a **second terminal window** and launch the desktop application:

python gui.py
