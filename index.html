Under the Hood: RESTful Architecture

Unlike traditional monolithic apps, Pustaklok separates concerns. The frontend is purely a consumer of the backend API, ensuring scalability.

  • Flask Backend: Handles logic, validation, and JSON storage.
  • Tkinter Frontend: Native desktop performance, zero web lag.
  • HTTP Requests: Standardized JSON communication.
